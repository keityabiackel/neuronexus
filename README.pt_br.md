# neuronexus

Uma aplicação web baseada em navegador para famílias neurodivergentes gerenciarem dados operacionais compartilhados, estados do ambiente doméstico e contextos de cuidado.  
Projetada para substituir check-ins emocionais subjetivos por informações observáveis e quantificáveis, utilizáveis no cotidiano, na educação, na saúde e em situações de emergência.

---

## NeuroNexus Hub

**NeuroNexus** é uma aplicação web executada no navegador, desenvolvida no âmbito do  
**PRISMAS – Núcleo de Estudo e Pesquisa Interdisciplinar em Autismo**.

O projeto foi concebido para apoiar famílias neurodivergentes por meio de dados compartilhados, objetivos e ancorados na realidade, priorizando clareza, previsibilidade e coordenação, em vez de check-ins emocionais subjetivos.

---

## Propósito

O NeuroNexus foi criado para enfrentar uma lacuna comum em aplicativos existentes voltados à saúde mental e ao cuidado:

A maioria das ferramentas foca no auto-relato individual, enquanto famílias neurodivergentes frequentemente necessitam de informações operacionais compartilhadas, compreensíveis entre casa, escola, saúde e contextos de emergência.

A ideia central é simples:

- menos interpretação  
- mais dados observáveis  
- uma linguagem operacional comum para todos os envolvidos  

---

## MVP Atual (Fase 0)

A versão atual é intencionalmente mínima e funciona inteiramente no navegador.

### Funcionalidades implementadas:

- **Estado da Casa (B):** indicador compartilhado do estado do ambiente doméstico (VERDE / AMARELO / VERMELHO)
- Persistência de estado usando armazenamento do navegador (LocalStorage)
- Arquitetura totalmente estática, executada no cliente
- Sem backend, autenticação ou banco de dados nesta etapa

Essa abordagem permite validação rápida dos conceitos, sem sobrecarga de infraestrutura.

---

## O Que Este Projeto Não É (Ainda)

- Não é uma ferramenta clínica
- Não é um sistema diagnóstico
- Não é uma plataforma completa de gestão de cuidados
- Não substitui o julgamento profissional

Essas questões pertencem a fases futuras, caso o projeto venha a evoluir.

---

## Roadmap (Visão Geral)

Possíveis extensões futuras incluem:

- Registro de eventos e janelas temporais (A)
- Marcação de contextos (família, escola, cuidado, emergência)
- Visualizações de acesso somente leitura para emergências
- Integração opcional com backend (ex.: Supabase)
- Controle de acesso baseado em papéis (famílias, educadores e profissionais)

Todo o desenvolvimento futuro seguirá os mesmos princípios: **modularidade, transparência e respeito à autonomia neurodivergente**.
